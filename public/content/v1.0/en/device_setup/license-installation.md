# License Installation

## Overview
This guide explains how to install licenses for using the RBWare system.

## License Types
- **Standard License**: Basic features
- **Professional License**: Advanced features included
- **Enterprise License**: Full features + technical support

## Installation Procedure

### 1. Prepare License File
Prepare the provided license file (.lic).

### 2. Register License
Register the license in system settings.
1. Access settings menu
2. Select license management
3. Upload file or enter key

### 3. Verify Activation
Confirm the license is properly activated.
- Status: `Active`
- Expiration date: Check contract period

## License Renewal
Renewal required before license expiration.
- Notification 30 days before expiration
- Renewal inquiry: support@rbware.co.kr

## Precautions
- Licenses are issued per device
- Unauthorized copying and distribution prohibited
- Reissuance required if license file is lost
