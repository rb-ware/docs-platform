# 장비 설치

## 개요
RBWare 시스템 장비의 설치 절차를 안내합니다.

## 설치 전 준비사항
- 전원 공급 장치 확인
- 설치 공간 확보 (최소 2m x 2m)
- 안전 장비 착용

## 설치 절차

### 1. 장비 배치
장비를 설치 위치에 배치합니다.
- 수평 상태 확인
- 고정 볼트 체결

### 2. 전원 연결
전원 케이블을 연결합니다.
- 전압 확인: AC 220V
- 접지 연결 필수

### 3. 초기 점검
설치 완료 후 기본 동작을 확인합니다.
- 전원 LED 확인
- 에러 메시지 확인

## 주의사항
- 설치 시 반드시 전원을 차단하세요
- 무거운 장비이므로 2인 이상 작업하세요
