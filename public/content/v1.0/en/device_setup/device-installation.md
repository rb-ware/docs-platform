# Device Installation

## Overview
This guide provides installation procedures for RBWare system equipment.

## Pre-installation Requirements
- Verify power supply
- Ensure installation space (minimum 2m x 2m)
- Wear safety equipment

## Installation Procedure

### 1. Device Placement
Position the device at the installation location.
- Check level surface
- Tighten fixing bolts

### 2. Power Connection
Connect the power cable.
- Verify voltage: AC 220V
- Ground connection required

### 3. Initial Inspection
Verify basic operation after installation.
- Check power LED
- Check for error messages

## Precautions
- Always disconnect power during installation
- Heavy equipment requires 2+ people for handling
