# 장비 간 통신 연결

## 개요
RBWare 시스템 장비 간 통신 네트워크 구성 방법을 설명합니다.

## 통신 방식
- Ethernet 기반 TCP/IP 통신
- 권장 속도: 1Gbps

## 연결 절차

### 1. 네트워크 구성
네트워크 토폴로지를 구성합니다.
- 스위치 준비
- 케이블 연결 (CAT6 이상 권장)

### 2. IP 주소 설정
각 장비의 IP 주소를 설정합니다.
```
기본 IP 대역: 192.168.1.x
서브넷 마스크: 255.255.255.0
게이트웨이: 192.168.1.1
```

### 3. 통신 테스트
Ping 테스트로 연결 상태를 확인합니다.
```bash
ping 192.168.1.100
```

## 문제 해결
- 연결 실패 시 케이블 연결 상태 확인
- 방화벽 설정 확인
- IP 충돌 여부 확인
