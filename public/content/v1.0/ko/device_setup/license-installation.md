# 라이선스 설치

## 개요
RBWare 시스템 사용을 위한 라이선스 설치 방법을 안내합니다.

## 라이선스 종류
- **Standard License**: 기본 기능 사용
- **Professional License**: 고급 기능 포함
- **Enterprise License**: 전체 기능 + 기술 지원

## 설치 절차

### 1. 라이선스 파일 준비
제공받은 라이선스 파일(.lic)을 준비합니다.

### 2. 라이선스 등록
시스템 설정에서 라이선스를 등록합니다.
1. 설정 메뉴 진입
2. 라이선스 관리 선택
3. 파일 업로드 또는 키 입력

### 3. 활성화 확인
라이선스가 정상적으로 활성화되었는지 확인합니다.
- 상태: `Active`
- 만료일: 계약 기간 확인

## 라이선스 갱신
라이선스 만료 전 갱신이 필요합니다.
- 만료 30일 전 알림
- 갱신 문의: support@rbware.co.kr

## 주의사항
- 라이선스는 장비별로 발급됩니다
- 무단 복제 및 배포 금지
- 라이선스 파일 분실 시 재발급 필요
